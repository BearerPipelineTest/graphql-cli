generate

Generate schema and client documents for your GraphQL project by using Graphback.

Installation#

  • yarn
  • npm
yarn global add @graphql-cli/generate
npm i -g @graphql-cli/generate

Example Configuration#

schema: './src/schema.graphql'
documents: './client/src/graphql/**/*.graphql'
extensions:
graphback:
model: './model/*.graphql'
plugins:
graphback-schema:
outputPath: './src/schema/schema.graphql'
graphback-client:
outputFile: './client/src/graphql/graphback.graphql'

See the docs for more details.

Usage#

graphql generate

Arguments#

None

Options#

optionaliasdescriptiondefault
--watch-wWatch for changes and execute generation automatically
--db
--backend
--silent
